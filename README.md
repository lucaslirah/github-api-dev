## 📦 Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/lucaslirah/github-api-dev.git

# Acesse a pasta do projeto
cd github-api-dev

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```
